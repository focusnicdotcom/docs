Web server adalah perangkat lunak (software) atau perangkat keras (hardware) yang bertugas untuk menyimpan, mengelola, dan menyajikan konten situs web kepada pengguna melalui internet. Web server menerima permintaan dari browser (seperti Chrome, Firefox, dll.), memprosesnya, dan kemudian mengirimkan kembali halaman web yang diminta kepada pengguna. 

Contoh web server:

- **Apache:** Salah satu web server yang paling populer dan banyak digunakan di seluruh dunia, mendukung berbagai sistem operasi.
- **Nginx:** Web server yang dikenal karena kecepatan dan efisiensinya.
- **LiteSpeed:** Mampu menggantikan Apache tanpa perlu mengubah konfigurasi atau aplikasi.

Web server menyimpan semua file yang membentuk sebuah situs web, seperti file HTML, gambar, video, dan lain-lain. Ketika pengguna mengakses sebuah situs web, browser mengirimkan permintaan ke web server. Web server akan menerima dan memproses permintaan tersebut. Setelah memproses permintaan, web server akan mengirimkan konten yang diminta kepada browser pengguna dalam bentuk halaman web. Web server juga bertanggung jawab untuk mengelola koneksi antara pengguna dan situs web, memastikan semua permintaan dilayani dengan efisien.
